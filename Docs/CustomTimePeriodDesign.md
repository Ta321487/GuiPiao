## 交互说明

### 1. 时间轴编辑

拖动边界线调整时段:
```
   00  04  08  12  16  20  24
   │   │   │   │   │   │   │
   └───┘   └───┘   └───┘   │
    凌晨    上午    下午   晚上
```

- 鼠标悬停边界线时显示 ↔ 光标，拖动即可调整
- 拖动时实时显示当前时间提示
- 释放时自动吸附到整点或半小时

### 2. 时段列表操作

| 操作 | 说明 |
|-----|------|
| **编辑** | 打开编辑面板修改名称、颜色、时间范围 |
| **删除** | 删除该时段（自动合并到相邻时段） |
| **添加** | 在列表末尾添加新时段 |

### 3. 验证规则

- ✗ 时段不能重叠
- ✗ 时段不能有空隙（必须覆盖00:00-24:00）
- ✗ 时段名称不能重复
- ✗ 开始时间必须小于结束时间
- ✗ 最少需要2个时段，最多支持8个时段

### 4. 预设模板

| 模板名称 | 时段划分 |
|---------|---------|
| 4段 (凌晨/早/中/晚) | 00-06 / 06-12 / 12-18 / 18-24 |
| 6段 (细粒度) | 00-04 / 04-08 / 08-12 / 12-16 / 16-20 / 20-24 |
| 通勤模式 | 00-06 / 06-09 / 09-12 / 12-14 / 14-18 / 18-21 / 21-24 |
| 夜间模式 | 00-06 / 06-18 / 18-24 |

### 5. 数据存储格式

```json
{
  "ClassificationBasis": "自定义时段",
  "CustomTimePeriods": [
    {"Name": "凌晨", "StartHour": 0, "EndHour": 6, "Color": "#1976D2"},
    {"Name": "早高峰", "StartHour": 7, "EndHour": 9, "Color": "#D32F2F"},
    {"Name": "上午", "StartHour": 9, "EndHour": 12, "Color": "#388E3C"},
    {"Name": "下午", "StartHour": 12, "EndHour": 18, "Color": "#FBC02D"},
    {"Name": "晚高峰", "StartHour": 18, "EndHour": 20, "Color": "#D32F2F"},
    {"Name": "晚上", "StartHour": 20, "EndHour": 24, "Color": "#7B1FA2"}
  ]
}
```

## 技术实现要点

### 模型层

```csharp
public class CustomTimePeriod
{
    public string Name { get; set; } = string.Empty;
    public int StartHour { get; set; }
    public int StartMinute { get; set; }
    public int EndHour { get; set; }
    public int EndMinute { get; set; }
    public string Color { get; set; } = "#1976D2";
}
```

### 视图层

- 时间轴控件：使用 Canvas + Rectangle 绘制
- 拖动交互：MouseDown/MouseMove/MouseUp 事件
- 时间提示：Popup 或 ToolTip

### 业务逻辑

- 时段验证：确保连续无重叠
- 自动调整：删除时段时自动扩展相邻时段
- 默认填充：新增时段默认填充剩余时间
